## 🏆 Featured Project: NextGZ ERP

<div align="center">

[![NextGZ ERP](https://github-readme-stats.vercel.app/api/pin/?username=Omariko2611&repo=NextGZ-crm-erb-system&theme=tokyonight&hide_border=true&bg_color=0d1117)](https://github.com/Omariko2611/NextGZ-crm-erb-system)

</div>

**Enterprise Resource Planning System** for rubber industry export operations:

- 🚢 **Shipment Management** - Track containers, ports, weights
- 🚛 **Fleet Module** - Trucks, drivers, trips, fuel logs
- 👥 **HR System** - Employees, attendance, payroll, leaves
- 📦 **Warehouse** - Inventory, locations, stock moves
- 📊 **Reports & Analytics** - Dashboards, exports, KPIs
- 🔐 **Blue/Green Deployment** - Zero-downtime updates
- 🐳 **Dockerized** - Full containerization

**Live Demo:** [erp.barisrubber.com](https://erp.barisrubber.com)
